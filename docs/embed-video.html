Embed video

Youtube

If video_clicktoload is enabled, Youtube iframe should be blocked and replaced by an optimized iframe containing only an image, a title, a button and a link, no script (styled like original Youtube). A click should launch the video on youtube.com in a low quality.

note 1 : quality setting is only available on Youtube.com (via an hack)

note 2 : presets Light = 144p, Normal = 360p

Vimeo

If video_clicktoload is enabled, Vimeo iframe should be blocked and replaced by an optimized iframe containing only an image, a title, a button and a link, no script (styled like original Vimeo). A click should launch the video in a new tab (iframe url + quality parameter) in a low quality.

note 1 : quality setting is only available for embed url with quality parameter

note 2 : presets Light = 240p, Normal = 360p

Dailymotion

If video_clicktoload is enabled, Dailymotion iframe should be blocked and replaced by an optimized iframe containing only an image, a title, a button and a link, no script (styled like original Dailymotion). A click should launch the video in a new tab without quality optimisation :(

note : quality parameters doesn't work

Facebook

If video_clicktoload is enabled, Facebook iframe should be blocked and replaced by an optimized iframe containing only an image, a title, a button and a link, no script (styled like original Facebook). A click should launch the video in a new tab without quality optimisation :(

note : quality parameters is not available for facebook video

Twitch

Twitch embed is no longer supported. Twitch closed their oEmbed service.
